# Welcome to Beanie's Moons

Beanie's Moons is a mod that introduces 8 new Vanilla Plus themed moons into the game, with varying prices and difficulty levels, with some featuring unique twists!

All of the moons have Individual downloads too, which can be found on my profile. (also check them if you want to see more screenshots!)

If you have any feedback or bug reports, please contact me on discord, my username is Beaniebe! All feedback is greatly appreciated!

Thank you to explodingturtles456 for the new veld corp sign asset!

# 52-Desperation - Difficulty: Hard - 300 Cost

52-Desperation is a primarily mansion-moon dominated by bugs, masked men and old birds. It costs 300 to travel to, but it is dangerous and expansive inside. Are you up for the challenge? Additionally, item equipment such as ladders, shovels and flashlights can spawn inside.
