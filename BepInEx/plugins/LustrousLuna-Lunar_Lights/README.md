# Lunar Lights

<b>This pack contains 7 moons.</b>

Note: All the moons are also available as standalone mods, <b>[you can find them here](https://thunderstore.io/c/lethal-company/p/LunarLights/)</b>.
(Make sure you dont have any of the standalones installed when using this mod)
