This mod makes it so any door sockets with the tag ID 7999 can not connect. This is particularly useful in making an interior like Tower, where I use the door ID on the two sideways doors, forcing it to generate upwards. You technically don't need this actually in your project, but if you want to use it, I'd recommend it on an empty game object in the main scene it generates in.

In order to use it, take any doorways that you don't want to connect, create a new tag, and give them the ID 7779.

![Tags shown on the door socket script](https://i.ibb.co/ds0mKgVT/Tag.png)

Now whenever this tile generates, it cannot generate itself connecting via that doorway. Now if you know how tower works, both the doors to the left and right have the ID 7779, meaning they cannot connect the tile from the sides at all. 

![Showing with and without script generation](https://i.ibb.co/bjHC78jb/Tower.png)
